GNB (Global Navigation Bar)

모든 페이지 상단 고정 · 높이 56px
🎬 VOD 홈 스마트 추천 👤

GNB 인터랙션

요소동작비고
로고 클릭 → 홈 페이지 / 이동
홈 탭 클릭 → 홈 페이지 / 이동 현재 페이지면 스크롤 최상단
스마트 추천 탭 클릭 → 스마트 추천 /recommend 이동
마이페이지 아이콘 클릭 → 마이페이지 /my 이동
시리즈 상세 페이지는 GNB가 동일하게 표시되나, 탭 하이라이트는 없음. 포스터 카드 클릭으로만 진입 가능.

인터랙션

요소동작비고
X 닫기 버튼 → 이전 페이지 (router.back) 시리즈 상세 페이지로 복귀
구매 옵션 선택 → 선택 항목 파란색 테두리 하이라이트
버튼 텍스트 "NP 결제하기"로 업데이트		 라디오 버튼 방식 (단일 선택)
보유 포인트 표시 현재 userAccount.points 실시간 표시 구매 완료 후 차감 반영
NP 결제하기 버튼 포인트 충분: 차감 → point_history 기록 → 구매 완료 메시지
포인트 부족: "포인트가 부족합니다." 에러 표시		 완료 후 1.5초 뒤 시리즈 상세로 이동
구매 완료 후 시리즈 복귀 → 시리즈 상세에서 구매 상태 자동 갱신
버튼: "구매하기" → "▶ 1화 시청하기"		 purchasedIds Set에 series_id 추가됨

목적지 페이지진입 가능 경로
/ 홈 GNB 로고 클릭 · GNB 홈 탭 클릭 · 직접 URL
/recommend 스마트 추천 GNB 스마트 추천 탭 클릭
/series/:id 시리즈 상세 홈 히어로 배너 · 홈 포스터 카드 · 이어보기 카드 · 스마트 추천 포스터 · 마이페이지 각 탭 아이템 · 시리즈 상세 비슷한 콘텐츠
/purchase/:id 구매 시리즈 상세 "구매하기" 버튼 (미구매 상태에서만)
/my 마이페이지 GNB 마이페이지 아이콘 클릭
📌 Mock 데이터 기준 전역 상태
purchasedIds: 구매 완료 series_id Set (시청 내역 항목은 기구매로 초기화)
wishlistIds: 찜 Map<series_id, 찜한날짜>
episodeProgress: 에피소드별 진행률 Map<episode_id, {completion_rate, watched_at}>
userAccount.points: 포인트 잔액 (초기값 100,000P)
